5. request header(요청 헤더)

데이터포맷지정; API의 정보를 XML형식으로 불러올지, JSON형식으로 불러올지!

	Header name=Accept
	Header value= ⓐ xml로 응답 : application.xml
		       ⓑ json로 응답 : applicatoin.json	//json으로 쓰자-Key만 알면 Value를 알 수 있으므로

결과값을 이용할 수 있도록 파싱작업을 해야 함
JSON형태로 불러오려면, AJAX에서 JSON형태로 요청하고, JSON형태로 API를 불러온 후 JSON으로 파싱하여 리턴한다.
